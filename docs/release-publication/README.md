# Release Publication Kit

This directory is non-normative.

It provides illustrative material for publishing `AGRP v1.0.0` consistently.

These materials do not define the release boundary themselves.

The normative sources remain:

- `spec/versioning/agrp-v1-release-definition.md`
- `spec/charter/agrp-v1-artifact-set.md`

Use this kit to:

- assemble a release package or publication record
- verify that the release identifier and normative anchors are present
- avoid accidental omission of required anchor references

Do not use this kit to infer new release requirements beyond what the normative documents already define.
