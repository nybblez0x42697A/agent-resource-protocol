# Spec

Normative protocol writing belongs here.

- `charter/`: mission, scope, and non-goals
- `glossary/`: canonical protocol terms and neutral vocabulary
- `conformance/`: baseline conformance requirements
- `bindings/`: concrete transport bindings for protocol contracts
- `security/`: baseline security, authorization, and policy rules
- `extensions/`: extension and stricter profile rules
- `discovery/`: capability advertisement and negotiation rules
- `versioning/`: unified versioning and evolution rules
- `deprecation/`: deprecation and sunset rules
- `composition/`: dependency and composition rules
- `diagnostics/`: failure taxonomy and bounded diagnostic rules
- `observability/`: request correlation and trace continuity rules
- `rollout/`: staged enablement and rollout rules
- `governance/`: artifact stewardship and namespace governance rules
- `compliance/`: readiness profiles, declaration interoperability, resolution, precedence, and partial-failure rules
- `rspl/`: resource-substrate protocol drafts
- `sepl/`: self-evolution protocol drafts
- `control-plane/`: interface and operation definitions
- `deployment/`: shared and external deployment design
